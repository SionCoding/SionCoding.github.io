[108] 将有序数组转换为二叉搜索树

  • 将一个按照升序排列的有序数组，转换为一棵高度平衡二叉搜索树。
  • 本题中，一个高度平衡二叉树是指一个二叉树每个节点 的左右两个子树的高度差的绝对值不超过 1。
两个注意点
  • int mid = (left + right) / 2; 会产生溢出，所以应该用 int mid = left + (right - left) / 2; 代替
  • / 2 可以用 >> 1 向右移位代替
